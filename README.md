# JavaScript DOM & BOM Tasks

A structured repository dedicated to mastering the JavaScript Document Object Model (DOM) and Browser Object Model (BOM) through practical challenges and interactive web tasks developed during the **NTI Training Program (Open-Source Application Development Track)**.

## 📌 Project Overview
This repository contains a collection of frontend applications focusing on manipulating web pages dynamically, handling browser environments, and structuring clean JavaScript code.

### Key Features Covered:
* **Dynamic DOM Manipulation:** Real-time element generation, style updates, and dynamic DOM traversal.
* **BOM Interactivity:** Utilizing `window.location`, query strings, and parsing URL parameters via `URLSearchParams`.
* **Event Handling:** Implementing clean and optimized event listeners for absolute user interaction control.
* **Conditional Logic & Styling:** Dynamic interface changes (like changing document foreground/background colors) based on parsed user states and credentials.

## 📁 Repository Structure
* `day5/`: Interactive login/landing simulator utilising `URLSearchParams` to extract and validate `name` and `pass` parameters from the query string.

## 🛠️ Tech Stack
* **Core:** HTML5, CSS3, JavaScript (ES6+)
* **Environment:** Linux (Ubuntu) & Git
